If you use Safari or a browser that can display favicons and the current favicon is that gay little netscape one, then I know how to fix it. If you are using a browser other then Safari, head to your favorite search engine and figure out how to delete the favicon (or icon) cache.

If you are on Safari, do the following:
1. Quit Safari.
2. In the Finder, navigate to this folder: ~/Library/Safari
3. Delete the "icon" folder.
4. Restart Safari and you're good.
